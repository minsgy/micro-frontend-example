## MFA Example (webpack-module-federation)

---

해당 프로젝트는 webpack-module-federation을 이용한 마이크로 프론트엔드를 구현한 예제 프로젝트입니다.

### Project start

```bash
# 프로젝트 루트 디렉토리로 이동
cd examples/webpack-module-federation

# 의존성 설치
yarn set version 3.6.0
yarn install

# 프로젝트 실행
yarn turbo:start
```
